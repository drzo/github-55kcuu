Logicmoo CogServer

This is a CogServer-compatible network interface.

Connection Details

The server provides:

  • CogServer interface on port 17001
  • REST API on port 3000

Usage

Connect using telnet:

telnet localhost 17001

Available Shells

  • Enter scm for Scheme shell
  • Enter pl for Prolog shell
  • Enter sexpr for S-expression shell

Note: This is a server application that requires a direct TCP connection. The web interface is for informational purposes only.
